Операторы сравнения `==`, `!=`, `>`, `<`, `>=`, `<=`
---
---

Все операторы работают интуитивно понятно, если проверка ведется 
с числами 1 и 0 то работает это след образом, 1 всегда равен True,
а 0 всегда равен False.

```python
    print('4 < 5 = ', 4 < 5)
    print('10 > 5 = ', 10 > 5)
    print('7 <= 7 = ', 7 <= 7)
    print('0 >= 0 = ', 0 >= 0)
    print('3 == 3 = ', 3 == 3)
    print('3 == 3.0 = ', 3 == 3.0)

    print('1 == True = ', 1 == True)
    print('7 == True = ', 7 == True)
    print('0 == False = ', 0 == False)

    print('1 != 10 = ', 1 != 10)
    
    # Вывод
    # 4 < 5 =  True
    # 10 > 5 =  True
    # 7 <= 7 =  True
    # 0 >= 0 =  True
    # 3 == 3 =  True
    # 3 == 3.0 =  True
    
    # 1 == True =  True
    # 7 == True =  False
    # 0 == False =  True
    
    # 1 != 10 =  True
```
